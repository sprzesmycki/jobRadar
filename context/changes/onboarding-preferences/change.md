## Notes

Thin vertical slice after the first Cloudflare Workers deployment. The slice gives an authenticated user a first product loop: save job-search preferences, see matched demo offers from the target sources, and save an offer status. It intentionally does not implement real scraping, CV upload/parsing, AI scoring, or cover-letter generation yet.
